# AI Dataset Builder for Model Training

## Objective
Create a high-quality dataset for training LLMs.

## Features
- Curated code examples.
- Improved AI-generated code.
- Benchmarks for performance evaluation.

## Usage
1. Install dependencies: `pip install -r requirements.txt`.
2. Run benchmarks: `python benchmarks/sorting_benchmark.py`.
3. View curated examples in the `curated_code/` directory.

## Contribution
Feel free to contribute by submitting pull requests.
